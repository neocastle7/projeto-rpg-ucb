# 🛡️ RPG de Turnos - Grupo UCB

Este repositório contém a base do nosso trabalho de Programação Orientada a Objetos.
Cada integrante é responsável por uma parte específica do código.

## ⚠️ Regras Importantes (Do Professor)
* **Nomes Claros:** Use nomes descritivos como `pontosDeVida` ou `calcularDano`.
* **Simplicidade:** Funções pequenas e código limpo.
* **Honestidade:** Cada um deve programar sua parte para garantir o aprendizado.

## 👥 Atribuições do Grupo
* **Jennyfer (`Jogador.java`):** Criar atributos (vida, nível, ataque), funções de atacar, receber dano e sistema de evolução.
* **Luan (`Inimigo.java`):** Criar atributos dos monstros, o Boss final e o sistema de premiação.
* **Jackson (`Calculos.java`):** Implementar a classe `Random` para trazer sorte e aleatoriedade aos combates.
* **Leonardo (`Historia.java`):** Criar a narrativa no `Main` e os diálogos que conduzem o jogador.
* **Lucas:** Lógica de turnos (ataque/defesa/especial) e o diferencial visual de cenários (Extra).
